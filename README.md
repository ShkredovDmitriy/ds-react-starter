# ds-react-starter

A universal foundation for your next react based project.

## Usage

#### Clone the repository
```
git clone https://github.com/ShkredovDmitriy/ds-react-starter.git PROJECT-NAME
```

#### Installation
```
npm install
```

#### Start Dev Server
```
npm run dev
```

#### Build Prod Version
```
npm run export
```
